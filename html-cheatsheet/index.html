CSS Positions

Types of Position values

  • Static
  • Realtive
  • Absolute
  • Fixed
  • Sticky

Static

The default setting. Follow the flow of the webpage.

Relative

Relative follows the natural flow of the webpage. Can change the position by inserting a value for top, bottom, right, and left.

Absolute

Removed the natural follow of the webage. Is relative to the element closest to it. Can change the position by inserting value for top, bottom, right, and left.

Fixed

Stays in the same spot on the webpage, even when scrolling. Can often see this for the nav bar or advertisements.

Sticky

Follows the natual flow of the webpage until it reaches the nearest parent orchild element. Can change the position by inserting value for top, bottom, right, and left.
